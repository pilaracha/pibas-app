## PIBAS APP
### Análisis y prototipado de un componente de visualización para usuarios finales en el marco de una herramienta de autor para recolección y análisis de datos. Un caso de estudio con perspectiva de género.

#### Descripción del Proyecto
PIBAS es un componente de visualización diseñado para usuarios finales en el marco de una herramienta de autor dedicada a la recolección y análisis de datos, en consonancia con el proyecto anterior "DEHIA: Una plataforma liviana para definir y ejecutar actividades con intervención humana basada en workflows" [Arcidiacono, 2020]. La plataforma DEHIA permite a los usuarios definir y ejecutar actividades con intervención humana para la recolección y análisis de datos, y PIBAS extiende su arquitectura de microservicios con un visualizador que facilita un análisis integral y flexible de la información recopilada.

#### Objetivos del Proyecto
- Mejorar la Experiencia de Usuario: Facilitar a los usuarios finales el análisis de datos a través de visualizaciones efectivas e interactivas.

- Contribuir al Análisis de Género: Aplicar las visualizaciones en un proyecto institucional centrado en fortalecer protocolos y estrategias de intervención ante situaciones de violencia de género en el ámbito universitario.

- Promover la Inclusividad: Distinguir y abordar las "violencias por razones de género" de manera inclusiva, reconociendo a personas con identidades LGBTQI+.

#### Tecnologías Utilizadas
#### Frontend: React.js
React.js, una biblioteca de JavaScript de código abierto desarrollada por Facebook, fue empleada en PIBAS para crear una interfaz de usuario efectiva. Sus ventajas incluyen la creación sencilla de componentes reutilizables y la capacidad de manejar grandes conjuntos de datos sin comprometer el rendimiento.

#### Backend: Node.js con Express JS
Para el desarrollo del backend de la plataforma, se utilizó Node.js, un entorno de tiempo de ejecución de JavaScript. Node.js, junto con el framework Express JS, ofrece facilidad para crear servidores web, capacidad para manejar múltiples solicitudes simultáneamente y la posibilidad de integrar bibliotecas y módulos de terceros.

#### Uso en Proyecto Institucional
PIBAS se aplica en un proyecto institucional colaborativo entre las UDAs-Áreas y la Dirección de Género y Diversidad de la Prosecretaría de DDHH-UNLP. El enfoque se centra en fortalecer protocolos y estrategias de intervención ante la violencia de género, contribuyendo al análisis y desarrollo de recursos institucionales para visibilizar la problemática y trabajar hacia una universidad libre de violencias.

El proyecto institucional mencionado se utilizará para validar la solución construida para PIBAS, centrándose en aportar valor a los datos recopilados y en la visualización dentro de un conjunto de representaciones predefinidas.
